# PADRON-PERCEPCIONES-Y-RETENCIONES-IIBB
Excel  para automatizar la búsqueda y cruce de alícuotas de Retenciones y Percepciones IIBB

Herramientas utilizadas en Excel:
Utiliza Power Query para modelar y conectar padrones mensuales de alto volumen (CABA y Bs. As.) y VBA para permitir consultas dinámicas por CUIT, reduciendo drásticamente el tiempo de validación fiscal.

1. Descargar el Arctivo excel con extencion ".xlsm"
2. Ocionalmente puedes descargar la carpeta  "PADRONES" a modo de ejemplo de los tipos de archivos admitidos.
